# Welcome to Sindre API

**Transform meeting recordings into actionable insights with AI**

Sindre is an intelligent meeting processing and product management platform that automatically extracts tasks, insights, and strategic bets from your meeting content. Our REST API provides comprehensive access to meeting intelligence, task management, and integration capabilities.

## Technology Stack

Built with modern technologies for scale, reliability, and performance.

<AccordionGroup>
  <Accordion icon="server" title="Backend Architecture">
    - **Runtime**: Node.js 18+ with Express.js
    - **Database**: Supabase (PostgreSQL) with Row Level Security
    - **Authentication**: Supabase JWT tokens with role-based access control
    - **Architecture**: Clean MVC pattern with services layer
  </Accordion>
  <Accordion icon="brain" title="AI & Machine Learning">
    - **LLM**: OpenAI GPT-4o for content analysis
    - **Transcription**: OpenAI Whisper for audio processing  
    - **Vector Database**: Pinecone for semantic search
    - **Framework**: LangChain for conversation management
  </Accordion>
  <Accordion icon="cloud" title="Cloud Services">
    - **Storage**: Google Cloud Storage for audio files
    - **Tasks**: Google Cloud Tasks for background processing
    - **Deployment**: Docker-ready with production configurations
  </Accordion>
</AccordionGroup>
