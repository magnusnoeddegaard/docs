# Initiative Management API

The Initiative Management API provides comprehensive lifecycle management for strategic initiatives, including AI-powered strategic planning through Sindre, owner assignment, status tracking, and meeting connections.

## API Endpoints Overview

<CardGroup cols={2}>
  <Card title="Core Initiative Operations" icon="target">
    - `POST /api/initiatives` - Create initiative
    - `GET /api/initiatives/:initiativeId` - Get initiative details
    - `PUT /api/initiatives/:initiativeId` - Update initiative
    - `DELETE /api/initiatives/:initiativeId` - Delete initiative
  </Card>
  <Card title="Initiative Retrieval" icon="search">
    - `GET /api/initiatives/user-initiatives` - Get user's initiatives
    - `GET /api/initiatives/space-initiatives` - Get space initiatives
    - `GET /api/initiatives/all-user-initiatives` - Get all user initiatives
    - `GET /api/initiatives/statuses` - Get available statuses
  </Card>
  <Card title="Initiative Updates" icon="pen">
    - `PATCH /api/initiatives/:initiativeId/status` - Update status
    - `PATCH /api/initiatives/:initiativeId/owner` - Update owner
    - `PATCH /api/initiatives/:initiativeId/priority` - Update priority
    - `PATCH /api/initiatives/:initiativeId/done` - Mark as done
  </Card>
  <Card title="Meeting Connections" icon="link">
    - `POST /api/initiatives/:initiativeId/meetings` - Connect meeting
    - `DELETE /api/initiatives/:initiativeId/meetings` - Disconnect meeting
    - `GET /api/initiatives/:initiativeId/meetings` - Get connected meetings
    - `GET /api/initiatives/unconnected-meetings` - Get unconnected meetings
  </Card>
</CardGroup>


## Create Initiative

Create a new strategic initiative. Initiatives can be assigned to users or spaces.

<CodeGroup>

```javascript POST /api/initiatives
const initiative = await fetch('https://api.sindre.com/api/initiatives', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${SINDRE_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Launch new product line',
    description: 'Develop and launch our new AI-powered productivity suite',
    priority_id: 2,
    icon_id: 3,
    target_date: '2024-12-31',
    user_id: 'user-uuid-here',
    space_id: 'space-uuid-here'
  })
});

const initiativeData = await initiative.json();
```

```python
import requests

response = requests.post(
    'https://api.sindre.com/api/initiatives',
    headers={
        'Authorization': f'Bearer {SINDRE_API_KEY}',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'Launch new product line',
        'description': 'Develop and launch our new AI-powered productivity suite',
        'priority_id': 2,
        'icon_id': 3,
        'target_date': '2024-12-31',
        'user_id': 'user-uuid-here',
        'space_id': 'space-uuid-here'
    }
)

initiative = response.json()
```

```bash
curl -X POST https://api.sindre.com/api/initiatives \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Launch new product line",
    "description": "Develop and launch our new AI-powered productivity suite",
    "priority_id": 2,
    "icon_id": 3,
    "target_date": "2024-12-31",
    "user_id": "user-uuid-here",
    "space_id": "space-uuid-here"
  }'
```

</CodeGroup>

### Request Body

<ParamField body="name" type="string" required>
  Clear, descriptive initiative name
</ParamField>

<ParamField body="description" type="string">
  Detailed description of the strategic initiative
</ParamField>

<ParamField body="status_id" type="string" default="1">
  Initiative status ID (defaults to planning)
</ParamField>

<ParamField body="priority_id" type="string" default="1">
  Initiative priority ID: 1=none, 2=low, 3=medium, 4=high
</ParamField>

<ParamField body="icon_id" type="string" default="1">
  Initiative icon ID for visual representation
</ParamField>

<ParamField body="target_date" type="string">
  Target completion date in YYYY-MM-DD format (defaults to 30 days from creation)
</ParamField>

<ParamField body="user_id" type="string">
  User ID to assign this initiative to (for personal initiatives)
</ParamField>

<ParamField body="space_id" type="string">
  Space ID to assign this initiative to (for team initiatives)
</ParamField>

### Response

Returns the created initiative object with status `201 Created`.

## AI-Powered Strategic Planning

When an initiative is assigned to Sindre (using `ownerId: "sindre"`), the AI assistant automatically:

1. **Moves the initiative to "In Progress" status**
2. **Starts a background strategic planning conversation** 
3. **Provides strategic analysis including**:
   - Strategic breakdown of the initiative
   - Key milestones and success metrics
   - Potential risks and mitigation strategies
   - Stakeholder identification and engagement approach
   - High-level implementation roadmap
   - Resource requirements and dependencies

<Note>
The AI focuses on strategic planning and coordination rather than detailed tactical execution, making it ideal for high-level initiative development.
</Note>
